
# 🎟️ Event Registration System

### Smart Event Management Platform

**Register • Manage • Track**

**Java • JSP • Servlets • MySQL**

---

## 🔍 Overview

Online Event Registration System is a web-based application that enables administrators to efficiently manage events through a centralized platform. The system provides secure login functionality and complete event management operations including adding, updating, searching, listing, and deleting event records.

---

## ⚡ Features

| Feature                 | Description                                    |
| ----------------------- | ---------------------------------------------- |
| 🔐 Secure Login         | Administrator authentication system            |
| ➕ Add Events            | Create and store new event records             |
| ✏️ Update Events        | Modify existing event information              |
| 🔍 Search Events        | Find events quickly using search functionality |
| 📋 View Events          | Display all registered events                  |
| ❌ Delete Events         | Remove events from the system                  |
| 💾 Database Integration | MySQL-based data storage                       |
